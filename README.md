# react-native-onepay-hash

Hash secure key logic for onepay

## Installation

```sh
npm install react-native-onepay-hash
```

## Usage

```js
import OnepayHash from "react-native-onepay-hash";

// ...

const result = await OnepayHash.multiply(3, 7);
```
