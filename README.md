# react-native-onepay-hash

Hash secure key and generate pay url for onepay

## Installation

```sh
npm install react-native-onepay-hash
```

## Usage

```js
import OnepayHash from "react-native-onepay";

// ...

const result = await OnepayHash.generateURl(params);
```
